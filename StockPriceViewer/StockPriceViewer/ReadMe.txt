This is a simple console application to fetch the live Stock Prices from Yahoo finance.
The url takes a field parameter &f which is used to get the specific details, 
in this sample we only fetch the symbol name bid and ask using these fileds are represented by the charectars s n b a
Once the solution is compiled and the console is launched, key in the symbol for which the bid and ask needs to be fetched
for example type AAPL and press enter, the results will be queried in the back ground and then shown in the console
for more than one symbols enter comma separated values for example AAPL,GOOG,DIS etc
To exit the application type Exit and press enter
